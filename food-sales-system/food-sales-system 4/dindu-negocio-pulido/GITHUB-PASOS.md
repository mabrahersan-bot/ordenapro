# Subir Dindu a GitHub

## 1. Crear cuenta

Entra a:

```text
https://github.com
```

Crea una cuenta o inicia sesion.

## 2. Crear repositorio

En GitHub:

1. Toca el boton `+` arriba a la derecha.
2. Toca `New repository`.
3. En `Repository name` escribe:

```text
dindu
```

4. Elige `Public` para que Render lo pueda conectar facil.
5. Toca `Create repository`.

## 3. Subir archivos

En la pagina del repositorio nuevo:

1. Toca `uploading an existing file`.
2. Abre en tu computadora la carpeta:

```text
outputs/food-sales-system
```

3. Arrastra todos los archivos y carpetas de `food-sales-system` a GitHub.
4. Baja hasta el boton verde.
5. Toca `Commit changes`.

## 4. Conectar con Render

Despues vuelve a Render y crea:

- Un `Web Service` para el backend.
- Un `Static Site` para las apps.

Sigue la guia:

```text
RENDER-PASOS.md
```

## Nota

No subas solo el archivo ZIP a GitHub. GitHub necesita ver las carpetas y archivos sueltos.
